# da_portfolio
В этом репозитории собраны проекты, которые я выполнила на базе Яндекс.Практикум.

**Анализ рынка общественного питания Москвы**
eda_food_service_market_portfolio
Проведён анализ 8 406 заведений Москвы; результат измерен структурой рынка, уровнем конкуренции, долей сетевых заведений, рейтингами и медианным чеком по округам; исследование помогает выбрать перспективный формат, ценовой сегмент и локацию для открытия заведения.

**Оценка результатов A/B-теста интернет-магазина**
online_store_ab_test_analysis
Проанализированы результаты A/B-теста на выборке из 9 963 пользователей после очистки пересечений между группами. Конверсия тестовой группы выросла с 27,81% до 29,54%, однако целевой эффект эксперимента достигнут не был. Анализ позволяет принять обоснованное решение и избежать преждевременного внедрения новой версии продукта.

**Инвестиционный анализ рынка стартапов**
startup_market_investment_analysis
Проанализированы объёмы, динамика и возвратность финансирования стартапов по сегментам и типам инвестиций; результат измерен долями привлечённого капитала, количеством раундов и динамикой возвратности до 2014 года; исследование позволило определить Software, Cloud computing и Big data как перспективные направления и рекомендовать венчурное финансирование.

**Подготовка данных билетных продаж сервиса «Яндекс Афиша»**
yandex_afisha_ticket_sales_analysis
Объединены и подготовлены данные о заказах, мероприятиях и площадках в 81 регионе, 352 городах и более чем 3 000 площадок; качество данных оценено по пропускам, дубликатам, валютам и выбросам до 99-го процентиля; результат создаёт надёжную основу для анализа продаж, категорий мероприятий и пользовательского поведения.

**Анализ пользовательской активности сервиса «Яндекс Книги»**
yandex_books_user_activity_analysis
Проведён статистический анализ активности 8 296 пользователей из Москвы и Санкт-Петербурга. Сравнено среднее время использования сервиса и проверена гипотеза о различиях между регионами. Исследование показало, что наблюдаемая разница статистически незначима, что помогает избежать необоснованных продуктовых решений.
